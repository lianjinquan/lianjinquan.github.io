以管理员身份打开cmd，执行如下命令：mysqld –initialize –console, 执行完成后会输出root的默认初始密码,ZwBI-,;qg9Nu 就是初始密码

启动命令：net start mysql

登录本机mysl命令：mysql -u root -p
