VMware安装cnotos6

下载centos6.8

centos6.8 64位下载地址,点击CentOS-6.8-x86_64-bin-DVD1.iso下载

安装

  1. 点击VMware中的创建新的虚拟机
  2. 选择典型（推荐）
  3. 选择稍后安装操作系统
  4. 客户机操作系统选择Linux,版本选择CentOS6 64位
  5. 选择安装位置
  6. 磁盘容量默认20GB
  7. 点击完成，完成安装
  8. 安装后界面如下
  9. 点击虚拟机-设置
    • 选择IOS镜像
    • 网络适配器选择 NAT模式
  10. 点击开启此虚拟机，选择 Install or upgrand an existing system
  11. skip跳过
  12. 点击next
  13. 选择中文简体
  14. 默认选择美国式英语
  15. 选择基本存储设备
  16. 选择是，忽略所有数据
  17. 主机名默认即可
  18. 选择默认时区
  19. 设置root账号的密码，本次设置 flzx_3QC (飞流直下三千尺)
  20. 选择创建自定义布局
  21. 点击创建-生成分区选择标准分区
    • /boot分区,固定大小选择200MB
    • /swap分区,固定大小选择1000MB
    • /home分区,固定大小选择2000MB
    • /根分区最后分,选择使用全部可用空间
    • 最后分区如下
  22. 选择格式化
  23. 将修改写入磁盘
  24. 默认即可
  25. 软件包选择，这里因为是个人的安装，我选择Desktop桌面,服务器推荐安装 Minial 最小化或者Basic server基本服务器
    • Desktop桌面
    • Minimal Desktop最小化桌面
    • Minimal 最小化
    • Basic Server基本服务器
    • Database Server 数据库服务器
    • Web Server 网页服务器
    • Virtual Host 虚拟主机
    • software development workstation 软件开发工具站
  26. 等待安装之后，点击重新引导
  27. 进入欢迎页面
  28. 点击同意
  29. 暂时忽略用户的创建
  30. 默认即可!
  31. kdump可以不勾选

  32. 登录桌面



  33. 配置网络连接，可以上网 System eth0
